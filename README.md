# File Storage Server

![License](https://img.shields.io/badge/License-MIT-green) ![Donations](https://img.shields.io/badge/Donations-orang)

File Storage Server - это веб-приложение для организации файлового хранилища с доступом через локальную сеть. Предназначено для быстрого обмена файлами между устройствами в одной сети без необходимости использования облачных сервисов.


## Что это

Этот сервер позволяет:
- Создавать защищенные паролем хранилища на любых папках вашего компьютера
- Загружать и скачивать файлы через браузер с любого устройства в сети
- Просматривать медиафайлы (изображения, видео) прямо в браузере
- Слушать музыку с отображением обложек и метаданных
- Редактировать текстовые документы и код прямо в веб-интерфейсе

Основная цель - превратить ваш компьютер в центральный файловый сервер для всей локальной сети.


## Для чего это нужно

- Домашняя сеть - обменивайтесь файлами между ноутбуком, ПК, телефоном и телевизором без USB-флешек
- Офис / Работа - создайте общее хранилище для сотрудников в локальной сети
- Медиасервер - просматривайте фильмы и слушайте музыку с любого устройства
- Резервное копирование - загружайте важные файлы на сервер с телефона или планшета
- Обмен с гостями - дайте доступ к файлам гостям в вашей сети без установки приложений


## Как это работает

1. Запустите сервер на компьютере: python app.py
2. Подключитесь с любого устройства в той же сети через браузер
3. Создайте хранилище - укажите папку на сервере
4. Загружайте файлы, просматривайте, скачивайте


## Установка и запуск

Требования:
- Python 3.6 или выше
- Установленные пакеты: flask, mutagen

### Установка:

1. Клонируйте или скачайте проект
   git clone https://github.com/yourusername/file-storage-server.git
   cd file-storage-server

2. Установите зависимости
   pip install -r requirements.txt

3. Запустите сервер
   python app.py

## Доступ к серверу

После запуска сервер доступен по адресам:

```bash
http://localhost:8000 # только с этого же компьютера
http://[IP-адрес-компьютера]:8000 # любого устройства в локальной сети
# IP-адрес сервера отобразится в консоли после запуска приложения
# IPCONFIG для просмотра IP вашего компьютера.
```

## Поддерживаемые устройства

- Windows, macOS, Linux - запуск сервера
- Android, iOS `- доступ через `браузер
- Умные телевизоры - просмотр медиа через браузер
- Любое устройство **с браузером в локальной** сети


## Возможности

Управление файлами:
- Загрузка файлов (перетаскиванием или через кнопку)
- Скачивание файлов
- Удаление файлов и папок
- Навигация по папкам с хлебными крошками

Просмотр медиа:
- Просмотр изображений (JPG, PNG, GIF, WEBP, SVG, BMP)
- Воспроизведение видео (MP4, WebM, AVI, MOV, MKV)
- Аудио-плеер с метаданными и обложками

Аудио-поддержка:
Поддерживаются форматы:
- MP3 (с обложками)
- FLAC (с обложками)
- OGG / OGA
- M4A / MP4 (с обложками)
- WAV
- AIFF / AIF

Редактор текста:
- Редактирование текстовых файлов (TXT, MD, PY, JS, HTML, CSS, JSON и др.)
- Подсветка и форматирование
- Горячие клавиши (Ctrl+S для сохранения)


## Безопасность

- Каждое хранилище защищено отдельным паролем
- Пароли хранятся в зашифрованном виде (SHA-256)
- Сессия активна только для одного хранилища за раз
- Работает только в локальной сети (по умолчанию)

Важно: Для доступа из интернета требуется настройка безопасности (HTTPS, firewall). Сервер не предназначен для публичного доступа без дополнительной защиты.
